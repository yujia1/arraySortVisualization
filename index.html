The Importance of Time Complexity

Time complexity is the amount of time it takes for a computer to run an algorithm. Good software engineering practice involves writing efficient algorithms, as doing otherwise can have serious consequences for both the developer and the client.

Time complexity is measured in "Big O" notation, and there are certain standards in the software engineering world that developers are supposed to work with. The purpose of this site is to visually show you the time complexity of four well known algorithms. More information on each can be read using the links to the left.

Time Complexities

  • Selection Sort: O(n2)
  • Insertion Sort: O(n2)
  • Merge Sort: O(n * log(n))
  • Quick Sort: O(n2)
